Banh-mi

题面翻译

题目描述

JATC很喜欢吃越式法包(一种越南食品)。他总是把它当早餐吃，因为他实在是太喜欢了。这天早上，像以往一样，他买了一个越式法包，并且决定以一种特殊的方法吃掉它。
首先，他把越式法包分为nnn块，并把它们排成一行，从111到nnn标号。对于每一块，他定义第iii块的_口感_为xi∈{0,1}x_i∈\{0,1\}xi​∈{0,1}。JATC正准备将它们一个接一个地吃掉。每一次，他随意选择剩下的一块吃掉。比如说他选择了第iii块，那么他的_愉悦度_就会增加xix_ixi​，并且所有剩下的块的_口感_也会增加xix_ixi​。最初，JATC的_愉悦度_等于0。
举个例子，假设333块越式法包的_口感_分别为[0,1,0][0,1,0][0,1,0]。如果JATC先吃掉第二块，他的_愉悦度_会变为111，其余块的_口感_则变为[1,_,1][1,\_,1][1,_,1]。接下来如果他吃掉第一块，他的_愉悦度_会变为222，剩下的块的_口感_变为[_,_,2][\_,\_,2][_,_,2]。吃掉最后一块后，JATC的_愉悦度_变为444。
然而，他不想吃掉所有的越式法包块儿，想留一些以后吃。他给了你nnn个询问，每个询问由两个整数lil_ili​和rir_iri​组成。对于每个询问，请告诉他当他以某种顺序吃掉区间[li,ri][l_i,r_i][li​,ri​]的所有块后，最大的愉悦度是多少。
每个询问都是互相独立的。由于答案可能很大，请对109+710^9+7109+7取模。

输入格式

第一行包含两个整数nnn和q(1≤n,q≤100 000)q(1\le n,q\le 100\ 000)q(1≤n,q≤100 000)。
第二行是一个长度为nnn的由’0’或’1’组成的字符串。第iii个字符表示了第iii块的口感。
之后的qqq行，每一行有两个整数lil_ili​和ri(1≤li≤ri≤n)r_i(1\le l_i\le r_i\le n)ri​(1≤li​≤ri​≤n)——每个询问的区间。

输出格式

输出qqq行，每行一个整数——答案对109+710^9+7109+7取模之后的结果。

样例1说明

对第1个询问：一种最佳的方案顺序为：1,4,3,21,4,3,21,4,3,2。
对第2个询问：以3,43,43,4或4,34,34,3的顺序都可以得到同样的答案。

样例2说明

任何顺序都能够得到相同的答案。

题目描述

JATC loves Banh-mi (a Vietnamese food). His affection for Banh-mi is so much that he always has it for breakfast. This morning, as usual, he buys a Banh-mi and decides to enjoy it in a special way.

First, he splits the Banh-mi intonnnparts, places them on a row and numbers them from111throughnnn. For each partiii, he defines the deliciousness of the part asxi∈{0,1}x_i \in \{0, 1\}xi​∈{0,1}. JATC’s going to eat those parts one by one. At each step, he chooses arbitrary remaining part and eats it. Suppose that part is theiii-th part then his enjoyment of the Banh-mi will increase byxix_ixi​and the deliciousness of all the remaining parts will also increase byxix_ixi​. The initial enjoyment of JATC is equal to000.

For example, suppose the deliciousness of333parts are[0,1,0][0, 1, 0][0,1,0]. If JATC eats the second part then his enjoyment will become111and the deliciousness of remaining parts will become[1,_,1][1, \_, 1][1,_,1]. Next, if he eats the first part then his enjoyment will become222and the remaining parts will become[_,_,2][\_, \_, 2][_,_,2]. After eating the last part, JATC’s enjoyment will become444.

However, JATC doesn’t want to eat all the parts but to save some for later. He gives youqqqqueries, each of them consisting of two integerslil_ili​andrir_iri​. For each query, you have to let him know what is the maximum enjoyment he can get if he eats all the parts with indices in the range[li,ri][l_i, r_i][li​,ri​]in some order.

All the queries are independent of each other. Since the answer to the query could be very large, print it modulo109+710^9+7109+7.

输入格式

The first line contains two integersnnnandqqq(1≤n,q≤100 0001 \le n, q \le 100\,0001≤n,q≤100000).

The second line contains a string ofnnncharacters, each character is either ‘0’ or ‘1’. Theiii-th character defines the deliciousness of theiii-th part.

Each of the followingqqqlines contains two integerslil_ili​andrir_iri​(1≤li≤ri≤n1 \le l_i \le r_i \le n1≤li​≤ri​≤n) — the segment of the corresponding query.

输出格式

Printqqqlines, whereiii-th of them contains a single integer — the answer to theiii-th query modulo109+710^9 + 7109+7.
