Nastya and a Wardrobe

题面翻译

题目描述

NastyaNastyaNastya 收到了一个神奇的“双倍衣柜”，在每个月底，衣柜中的裙子数量都会双倍。

在第 111 年里，衣柜中有 xxx 条裙子。前 kkk 个月中，在双倍之后，NastyaNastyaNastya 都有 50%50\%50% 的几率从衣柜中取出一条裙子（当然首先要保证衣柜内有裙子）。身为程序员，请你帮她求出第 k+1k + 1k+1 个月底，衣柜中的裙子双倍后，裙子数的期望值。这个值可能很大，你只要输出它对 109+710^9+7109+7 取模的结果即可。

输入格式

仅 111 行，有 222 个整数，分别为衣柜中原有的裙子数 xxx，以及给定的 kkk。

（数据范围：0⩽x, k⩽10180 \leqslant x,\ k \leqslant 10^{18}0⩽x, k⩽1018）

##输出格式

仅 111 个整数，表示第 k+1k + 1k+1 个月底衣柜中裙子数的期望值对 109+710^9+7109+7 取模的结果。

##提示与说明

  • 第 111 组样例的解释：

因为 k=0k = 0k=0，所以 NastyaNastyaNastya 无法取走任何裙子。因此期望值就是 xxx 双倍的结果，也就是 444。

  • 第 222 组样例的解释：

第一个月底双倍之后，衣柜中有 444 条裙子，NastyaNastyaNastya 有 50%50\%50% 的几率取走 111 条，这意味着衣柜中的裙子有 50%50\%50% 的几率有 333 条，50%50\%50% 的几率有 444 条。第二个月底她不会取裙子，因此此时衣柜中的裙子有 50%50\%50% 的几率有 666 条，50%50\%50% 的几率有 888 条。综上所述，期望值为 (6+8)/2=7(6+8)/2 = 7(6+8)/2=7。

感谢@Sooke 提供翻译

题目描述

Nastya received a gift on New Year — a magic wardrobe. It is magic because in the end of each month the number of dresses in it doubles (i.e. the number of dresses becomes twice as large as it is in the beginning of the month).

Unfortunately, right after the doubling the wardrobe eats one of the dresses (if any) with the5050%50probability. It happens every month except the last one in the year.

Nastya ownsxxxdresses now, so she became interested in the expected number of dresses she will have in one year. Nastya lives in Byteland, so the year lasts fork+1k+1k+1months.

Nastya is really busy, so she wants you to solve this problem. You are the programmer, after all. Also, you should find the answer modulo109+710^{9}+7109+7, because it is easy to see that it is always integer.

输入格式

The only line contains two integersxxxandkkk(0<=x,k<=10180<=x,k<=10^{18}0<=x,k<=1018), wherexxxis the initial number of dresses andk+1k+1k+1is the number of months in a year in Byteland.

输出格式

In the only line print a single integer — the expected number of dresses Nastya will own one year later modulo109+710^{9}+7109+7.
