Rectangles

题面翻译

题目大意

在平面直角坐标系中给定nnn个矩形，要求找到一个点(x,y)(x,y)(x,y)，这个点在至少n−1n-1n−1个给定矩形的内部或边界

输入格式

第一行一个整数nnn，表示矩形个数

下面nnn行每行444个整数x1,y1,x2,y2x1,y1,x2,y2x1,y1,x2,y2，表示矩形的左下和右上坐标

输出格式

两个整数x,yx,yx,y

题目描述

You are givennnnrectangles on a plane with coordinates of their bottom left and upper right points. Some(n−1)(n-1)(n−1)of the givennnnrectangles have some common point. A point belongs to a rectangle if this point is strictly inside the rectangle or belongs to its boundary.

Find any point with integer coordinates that belongs to at least(n−1)(n-1)(n−1)given rectangles.

输入格式

The first line contains a single integernnn(2≤n≤132 6742 \le n \le 132\,6742≤n≤132674) — the number of given rectangles.

Each the nextnnnlines contains four integersx1x_1x1​,y1y_1y1​,x2x_2x2​andy2y_2y2​(−109≤x1<x2≤109-10^9 \le x_1 < x_2 \le 10^9−109≤x1​<x2​≤109,−109≤y1<y2≤109-10^9 \le y_1 < y_2 \le 10^9−109≤y1​<y2​≤109) — the coordinates of the bottom left and upper right corners of a rectangle.

输出格式

Print two integersxxxandyyy— the coordinates of any point that belongs to at least(n−1)(n-1)(n−1)given rectangles.
